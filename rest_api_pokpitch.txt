Select a door

- PUT /{game_id}/door/0
{ “selected”: true }

- return
{
	“id”:”0”,
	“cat”: “FOUNDED”,
	“state”: “CLOSED”,
  “selected”: true
}

—————————————————

Open a door
- PUT /{game_id}/door/0
{ “state”: “OPENED” }
- return
{
	“id”:”0”,
	“cat”: “FOUNDED”,
	“state”: “OPENED”,
  “selected”: true
}
